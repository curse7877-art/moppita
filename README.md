# Moppita Ebook Capture

Landing page em React + Vite com API serverless para Vercel e banco no Supabase.

## Stack

- Frontend: React + Vite
- API: Vercel Function em `api/leads.ts`
- Banco: Supabase (tabela `public.leads`)

## Requisitos

- Node.js 20+

## Rodar local

1. Instale as dependencias:
   `npm install`
2. Configure as variaveis em `.env` (base no `.env.example`)
3. Rode:
   `npm run dev`

Servidor local: `http://localhost:5173`

## Deploy na Vercel

1. Suba o projeto no GitHub/GitLab/Bitbucket.
2. Importe o repositorio na Vercel.
3. Em `Environment Variables`, configure:
   - `SUPABASE_URL`
   - `SUPABASE_SERVICE_ROLE_KEY`
   - `ADMIN_USER` (opcional)
   - `ADMIN_PASSWORD` (opcional)
4. Deploy.

`vercel.json` ja esta configurado com:
- build command `npm run build`
- output `dist`
- rewrite para SPA (`/admin` funcionar em refresh)

## Estrutura da tabela no Supabase

```sql
create table if not exists public.leads (
  id bigint generated by default as identity primary key,
  name text not null,
  email text not null,
  phone text not null,
  created_at timestamp with time zone default now()
);
```

## Endpoints

- `POST /api/leads` - cria lead
- `GET /api/leads` - lista leads (Basic Auth)
